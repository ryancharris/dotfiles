# zsh

Key bindings and aliases defined in `.zshrc`.

## Key Bindings

| Key | Action |
|-----|--------|
| `alt+→` | Forward word |
| `alt+←` | Backward word |
| `↑` | History substring search up |
| `↓` | History substring search down |
