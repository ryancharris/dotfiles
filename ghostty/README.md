# ghostty

Custom keybindings defined in `config`. Main modifier is `cmd`.

## Config

| Key | Action |
|-----|--------|
| `cmd+shift+r` | Reload config |

## Splits

| Key | Action |
|-----|--------|
| `cmd+shift+h` | New split (below) |
| `cmd+shift+v` | New split (right) |
| `cmd+h` | Focus split left |
| `cmd+j` | Focus split down |
| `cmd+k` | Focus split up |
| `cmd+l` | Focus split right |

## Tabs

| Key | Action |
|-----|--------|
| `cmd+w` | Close tab |
| `cmd+shift+.` | Move tab forward |
| `cmd+shift+,` | Move tab backward |
