# nvim

Custom keymaps defined in `lua/config/keymaps.lua`. Leader key is `<Space>`.

## Buffers

| Key | Action |
|-----|--------|
| `<leader><Tab>` | Toggle most recent buffer |
| `<leader>wd` | Delete buffer |
| `<leader>bb` | Fuzzy search open buffers |

## Editing

| Key | Action |
|-----|--------|
| `<leader>rr` | Redo |
| `<leader>uu` | Undo |

## Files

| Key | Action |
|-----|--------|
| `<leader>ff` | Fuzzy search files |
| `<leader>ee` | Open file explorer (netrw) |
| `<leader>re` | Reopen file explorer |

## Git / GitHub

| Key | Action |
|-----|--------|
| `<leader>gb` | Git blame |
| `<leader>gc` | Git commits |
| `<leader>gs` | Git status |
| `<leader>gg` | Open current file in GitHub |

## LSP

| Key | Action |
|-----|--------|
| `<leader>cd` | Go to definition |
| `<leader>cr` | Go to references |
| `<leader>co` | Open diagnostic float |

## Search

| Key | Action |
|-----|--------|
| `<leader>ps` | Search word under cursor |
| `<leader>ss` | Live grep |

## Windows

| Key | Action |
|-----|--------|
| `<leader>wv` | Vertical split |
| `<leader>ws` | Horizontal split |
| `<leader>wh` | Focus window left |
| `<leader>wl` | Focus window right |
| `<leader>wj` | Focus window down |
| `<leader>wk` | Focus window up |
