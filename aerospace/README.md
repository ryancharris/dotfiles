# aerospace

Custom keybindings defined in `aerospace.toml`. Main modifier is `alt` (Option), keeping `cmd` free for app shortcuts.

## Focus

| Key | Action |
|-----|--------|
| `alt-h` | Focus window left |
| `alt-j` | Focus window down |
| `alt-k` | Focus window up |
| `alt-l` | Focus window right |

## Move Windows

| Key | Action |
|-----|--------|
| `ctrl-alt-h` | Move window left |
| `ctrl-alt-j` | Move window down |
| `ctrl-alt-k` | Move window up |
| `ctrl-alt-l` | Move window right |

## Split Direction

| Key | Action |
|-----|--------|
| `alt-shift-h` | Split horizontal (side-by-side) |
| `alt-shift-v` | Split vertical (stacked) |

## Layout & Fullscreen

| Key | Action |
|-----|--------|
| `alt-u` | Toggle fullscreen |
| `alt-shift-u` | Toggle floating / tiling |
| `alt-semicolon` | Cycle layout (tiles horizontal/vertical) |
| `alt-p` | Accordion layout |
| `alt-shift-semicolon` | Reload config |

## Workspaces

| Key | Action |
|-----|--------|
| `alt-1` … `alt-9` | Switch to workspace 1–9 |
| `alt-0` | Switch to workspace 10 |
| `alt-shift-1` … `alt-shift-9` | Move window to workspace 1–9 |
| `alt-shift-0` | Move window to workspace 10 |

## Resize Mode

Enter with `alt-r`. Press `enter` or `esc` to exit.

| Key | Action |
|-----|--------|
| `h` / `l` | Resize width ±50 |
| `k` / `j` | Resize height ±50 |
| `-` / `=` | Smart resize ±50 |
