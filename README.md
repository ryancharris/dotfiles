# dotfiles

## setup

- clone repository to local machine
- install cli tools and gui apps with [homebrew](https://brew.sh/) scripts
- setup symlinks

## tools

### frequently used

- [git](https://www.git-scm.com/)

- [tmux](https://github.com/tmux/tmux/wiki)
